# Rupee Radar

Rupee Radar is an Android expense tracker that reads transaction-related SMS messages on-device, classifies likely financial activity, and turns them into a local expense timeline and monthly summary.

## What it does

- imports transaction-like SMS messages from the device
- classifies messages on-device
- stores transactions locally
- shows monthly totals and grouped transaction history
- supports manual transaction entry and edits

## Privacy

Rupee Radar reads SMS only after permission is granted. Based on the current app build, processing happens on-device and the app does not request the `INTERNET` permission.
