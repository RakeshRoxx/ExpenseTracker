What it does

  • Reads transaction-like SMS after permission is granted
  • Classifies likely financial activity on-device
  • Saves transactions locally for summaries and review

Privacy

  • No `INTERNET` permission in the current app build
  • Transaction data stays on the device
  • Read the full policy before installing

Install notes

  • Download the APK from the latest GitHub Release
  • Allow installation from your browser or files app if Android asks
  • Grant SMS permission inside the app to import transactions
