# Rupee Radar v1.0.1

Early tester release for Rupee Radar.

## Highlights

- imports and classifies transaction-related SMS messages on-device
- stores transactions locally and shows monthly summaries
- supports manual transaction entry, edits, and deletion
- limits initial setup import to the current month for a faster first run
- includes a signed APK with a stable release filename for easier sharing

## Download

- APK asset: `RupeeRadar-v1.0.1-2-release.apk`

## Notes

- Android permissions are required for SMS import and notifications.
- Based on the current app build, the app does not request the `INTERNET` permission.
- Privacy policy: `https://rakeshroxx.github.io/ExpenseTracker/privacy-policy.html`
