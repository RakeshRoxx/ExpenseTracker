# Screenshot Checklist

Store these files in this folder once captured.

Recommended phone screenshots for Rupee Radar:

1. `01-dashboard-overview.png`
   Show the dashboard with the monthly total and category pie chart.

2. `02-grouped-transactions.png`
   Show the transaction list grouped by month with a few realistic entries.

3. `03-add-transaction.png`
   Show the manual add transaction dialog or sheet.

4. `04-transaction-details.png`
   Show the SMS preview or transaction details dialog.

5. `05-empty-state-or-refresh.png`
   Show the empty state or pull-to-refresh state to explain SMS import behavior.

Capture tips:

- Use a clean demo dataset with non-sensitive entries.
- Prefer portrait screenshots.
- Keep status bar time and battery consistent if possible.
- Avoid showing real SMS content, phone numbers, or account details.
- Export PNG if possible.
